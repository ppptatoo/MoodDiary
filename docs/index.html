✨ 核心功能

离线优先 · 云端同步 · AI 赋能

☁️

智能双存储模式

断网离线时数据仅存本地 Room 数据库，联网后自动与 Supabase PostgreSQL 双向同步。同账号换手机一键拉取全部历史日记，无缝切换。

📊

情绪数据统计

自定义环形饼图直观展示各类情绪占比，柱状走势图呈现每日情绪强度变化。 本月记录天数、平均情绪强度等量化统计，AI 自动生成情绪趋势洞察。

🤖

火山引擎 AI 智能

接入豆包大模型：根据日记内容自动生成 AI 治愈短句；一键生成插画配图并上传 Supabase Storage；多轮 AI 对话情绪陪伴助手。

🔐

数据安全隔离

不同账号数据完全独立，Supabase RLS 策略保障云端隔离，JWT 认证 + SharedPreferences 缓存登录态。敏感密钥通过 local.properties 注入，不提交 Git。

🎨

主题与个性化

暖色 / 深色主题一键切换，全局即时生效。头像更换支持相册选择与拍照， 自动圆形裁剪。个人资料编辑：昵称、邮箱、签名、密码修改。

🔔

智能通知提醒

每日心情记录提醒、每周情绪回顾报告、随机正能量鼓励推送， 全部支持自定义提醒时间。

🚀 三步运行

无需填写密钥即可纯本地运行

1

克隆仓库 & 配置密钥模板

git clone https://github.com/ppptatoo/MoodDiary.git
cp local.properties.example local.properties

2

（可选）填入密钥

编辑 local.properties 填入 Supabase 和火山引擎密钥。
不填也可以：应用自动以纯本地模式运行，AI 按钮隐藏。

3

Android Studio 打开 → Run

Sync Gradle，连接手机或模拟器，点击 Run ▶ 即可运行。
